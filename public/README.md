# Public Assets

このディレクトリには、サイトの公開アセットを配置してください。

## 必要なファイル

### 1. OGイメージ (`og-image.png`)
- **用途**: ソーシャルメディアでシェアされた際のプレビュー画像
- **推奨サイズ**: 1200 x 630px
- **形式**: PNG
- **配置場所**: `public/og-image.png`

### 2. ファビコン
Next.js App Routerでは、以下のファイルを `src/app/` に配置することで自動的にファビコンが設定されます：

- `src/app/favicon.ico` - 基本的なファビコン
- `src/app/icon.png` - 高解像度アイコン（推奨サイズ: 512 x 512px）
- `src/app/apple-icon.png` - Apple デバイス用アイコン（推奨サイズ: 180 x 180px）

## その他の推奨アセット

- `robots.txt` - 自動生成（`src/app/robots.ts`）
- `sitemap.xml` - 自動生成（`src/app/sitemap.ts`）

## 画像作成のヒント

### OGイメージのデザイン
- サイトのブランドカラー（ネオンシアン、ネオンパープル）を活用
- キャッチフレーズ「未来は予測するものではない。生き抜くものだ。」を含める
- 視認性の高いタイポグラフィ
- 暗い背景（#0f172a）でサイバーパンク風のデザイン

### ファビコンのデザイン
- シンプルで認識しやすいロゴ
- 「AI」の文字またはサイトを象徴するアイコン
- グラデーション（ネオンシアン to ネオンパープル）を活用
